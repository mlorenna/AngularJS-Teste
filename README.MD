# Project Title

A AngularJS Project for testing forms and tables.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

You will need to install angular 1.7.9
Run `npm install angular@1.7.9`

### Installing

Use `npm i to install` all packages and run `npm run copy-libs` to build our lib file

## Development server

Run `http-server ./app` for a server. Navigate to `http://localhost:8080`
